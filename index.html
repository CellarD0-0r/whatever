Whatever is an unofficial Evernote© client for Linux desktop

Open Source

Whatever is 100% free, non-commercial and independent project. It is not supported by Evernote© corporation or any other commercial organization.

Since Whatever is on active development stage, feel free to contribute and open issues on GitHub.

Built with Electron

Whatever is based on web-technologies and Electron framework.

Electron is an open source library developed by GitHub for building cross-platform desktop applications with HTML, CSS, and JavaScript. Electron accomplishes this by combining Chromium and Node.js into a single runtime. Therefore it provides web features and native integration at the same time.

Made for Linux

Whereas Evernote Corporation still doesn't support Linux platform, there is a 100% freedom for third-party developers to piece it out, as it is mentioned at the official Evernote support page.

Whatever provides native installers for the most popular Linux distributions. Also it can be built from the source code.
