Whatever Documentation

Built with Electron

Whatever is based on web-technologies and Electron framework.

Electron is an open source library developed by GitHub for building cross-platform desktop applications with HTML, CSS, and JavaScript. Electron accomplishes this by combining Chromium and Node.js into a single runtime. Therefore it provides web features and native integration at the same time.

Whatever is a wrapper running the web wersion in a separate window, but inspite of it's simplicity it provides desktop integration features like native installers, tray icon, context menu, notifications e.t.c.

Installing

Whatever is created for Linux and provides .deb and .rpm installers and .tar.gz package. Also it could be easilly built and installed from source code using electron-builder or electron-packager (check Application Distribution for more information.)



Open Source

Whatever is on active development stage, so please feel free to contribute and open issues on GitHub.
